**How to capture**: press `Cmd+Shift+4`, then `Space`, then click the app window — macOS's
window-capture mode bakes in a drop shadow, rounded corners, and a transparent background. A
flat screen-region crop (`Cmd+Shift+4` dragged) has none of that, so a white app window on
GitHub's white page background has no visible edge at all — that's the "white on white" look.
Window-capture fixes it without any extra editing, and it also reads fine on GitHub's dark theme.

Recommended: 720px wide, light mode, actual data (blur/redact hostnames or IPs you don't want public).
